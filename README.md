# 시험문제 출제 플랫폼

간호학과 교과목 시험문제를 AI로 출제·편집·인쇄하는 **단일 파일 웹 플랫폼**
청암대학교 간호학과 · 정종필 교수 · imjp5678@scjc.ac.kr

**▶ 바로 사용하기: https://jp5678.github.io/exam-authoring/**

---

## 주요 기능

- **학년도·학기 / 학년·교과목 / 시험 구분·일시** 선택 (교과목명 직접 입력, 시험 일시 캘린더+시각 드롭다운)
- **유형별 문항 수·배점**: 5지선다 객관식 · 참/거짓(진위형) · 단답형 · 주관식(서술형)
- **난이도 분포** 선택, **학습내용 입력 + 수업자료 파일 첨부**(docx·pdf·pptx·xlsx·hwpx 등 브라우저 내 텍스트 추출)
- **2가지 출제 방식**: Claude API 자동 출제 / 프롬프트 복사 출제(API 키 불필요)
- **실제 시험지 양식** 렌더링 + **정답·해설·채점기준** 자동 생성
- 문항 클릭 즉시 수정 · 문항/보기 섞기(A·B형) · 인쇄(PDF) · Word 저장 · 문제은행

자세한 소개는 [시험문제 출제 플랫폼 소개.md](시험문제%20출제%20플랫폼%20소개.md) 참고.

## 기술 스택

순수 HTML5 / CSS3 / Vanilla JavaScript 단일 파일 · Anthropic Claude API · JSZip · PDF.js · SheetJS

## 사용 방법

브라우저에서 위 주소로 접속하거나, `index.html`을 내려받아 더블클릭하면 설치 없이 바로 동작합니다.

> AI가 생성한 문항은 반드시 교수자가 검토 후 사용하세요.
